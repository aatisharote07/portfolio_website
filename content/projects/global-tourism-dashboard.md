## Project Overview

This project features an interactive global tourism dashboard visualizing trends across 232 countries with advanced data modeling. It provides a comprehensive view of global tourism statistics, enabling users to explore data at a macro and micro level.

## Key Highlights

- **Global Coverage**: Analyzes tourism data from 232 countries.
- **Advanced Data Modeling**: robust data models to ensure accurate and performant reporting.
- **Interactive Visualizations**: Users can drill down into specific regions, time periods, and metrics.
