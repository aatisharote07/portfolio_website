## Project Overview

Tourism AI is an AI-powered tourism analytics platform delivering real-time insights from dashboards and OCR-driven intelligence. This project integrates machine learning models with interactive visualizations to help stakeholders make informed decisions in the tourism sector.

## Key Features

- **Real-time Analytics**: Provides up-to-date insights into tourism trends and patterns.
- **OCR-driven Intelligence**: Utilizes Optical Character Recognition to extract and analyze data from various document types.
- **Interactive Dashboards**: Built with Power BI to offer granular views of data across different dimensions.
